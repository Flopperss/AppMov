Comando para instalar Cypress
npm install cypress --save-dev

Comando para ejecutar Cypress
npx cypress open

Comando para ejecutar el karma(Karma se instala automaticamente al crear nuestro proyecto)
ng test

Comando Para ejecutar el server local
ionic serve

Comando Para instalar json-server
npm install -g json-server

Falta el comando para ejecutar json-server Pero no me acuerdo mucho de su uso

Comando para libreria de qr
npm install jsqr

Instalar paquetes npm
npm install
